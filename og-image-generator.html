OG Share Image Generator

This generates the 1200×630 PNG used as the default Open Graph share card on Twitter, Facebook, LinkedIn, etc.

Click "Download og-default.png" below. Save it directly into the project root (next to index.html).

After downloading: commit + push the new og-default.png to GitHub. Cloudflare Pages auto-deploys it. The OG meta tags in index.html already point to it.
