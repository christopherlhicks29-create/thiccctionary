Quick-fire actions

Trigger any pipeline workflow without going through the Actions tab. Each click commits a sentinel file to main, which fires the corresponding workflow within ~30 seconds.

Generate today's daily entry

Triggers daily.yml. Use if the 13:00 UTC cron didn't fire or you want to retry. Won't duplicate if today's entry already exists (unless you delete it first).

Wave 220i: Force bypasses the “entry already exists” check. Use if the cron runs green but produces no entry (silent-skip mode).

Burst-create archive entries

Paste a list of subjects (one per line). Each runs the full daily pipeline with quality gate (image veto + critique ≥ 6/10) and back-fills the archive going backwards from the oldest existing entry. Max 25 subjects per burst. Auto-merges when successful.

Subject format: Word, Qualifier like Anvil, Peter Wright or Watermelon, Carolina Cross. A bare word works too.

Build Reel video

Triggers build-tiktok.yml via sentinel. Generates the deadpan-voice 1080×1920 MP4 + commits to videos/<date>.mp4.

Cross-post Reel to FB + IG

Triggers fire-reel.yml. Queues the Reel video in Buffer for FB Reels + IG Reels. Requires the MP4 to already exist on main.

Backfill morning post + newsletter

Triggers fire-backfill.yml. Re-runs morning Buffer post + Reel cross-post + Buttondown newsletter for a specific date. Use if post-on-merge failed and you need to ship those posts retroactively.

Brand mention monitor

Triggers brand-mention.yml. Polls Reddit + HackerNews for "thiccctionary" mentions. Writes audits/brand-mentions/<date>.md with new mentions highlighted at top. Scheduled daily 13:00 UTC. Free, no auth, $0/run.

Force editorial (From the Boat)

Triggers from-the-boat.yml. Bertram Whitmore, Founder & Publisher, writes a meandering dispatch from the Margaret IV. Scheduled 1st and 15th of every month. Saves to articles/from-the-boat-YYYY-MM-DD.html.

Image audit (catalog-wide)

Triggers image-audit.yml. Runs the Wave 204 critic against every entry's live image. Flags mismatches (caption says X, photo shows Y). Auto-queues a regen for the worst offender. Run weekly via cron, or fire on-demand.

Social audit

Triggers social-audit.yml. Scans the last 14 days of office posts, Buffer responses, and entry data. Surfaces broken captions, duplicates, and posts with hidden Buffer failures. Results appear on the Social audit tab + audits/social-audit/.

Buffer queue

Manage the Buffer scheduled-post queue directly via API. List dumps the queue contents to audits/buffer-queue/. Purge deletes posts whose text contains any of the match phrases (newline-separated, case-insensitive).

Fire an employee social post

Triggers office-post.yml. Generates a Claude-written, gpt-4o-rated (>=7/10) ~280-char post bylined by a staff member and posts to FB + X via Buffer (Instagram skipped, text-only). Random byline by default, or pick one. Topic kind defaults to "either".

Employee social posts

All staff-bylined posts from data/office-post-queue.json. Status = posted means Buffer accepted the post into its queue; the actual FB/X publish happens at the next Buffer scheduled slot (use Buffer dashboard to share-now if needed).
