# Internal resources

Files here are for Christopher's reference, not for public consumption.

## thiccctionary-roadmap.ics

Calendar file with recurring book-roadmap reminders:
- **Weekly:** Sunday 10am, curation pass (review past 7 entries, flip bookReady)
- **Monthly:** 1st of each month, 9:30am, book-status check
- **Quarterly:** 1st of Aug/Nov/Feb 2027, major recalibration
- **Phase milestones:** July 6 (Phase 2 / POD partner), Oct 5 (Phase 3 / layout tool), Dec 1 (Phase 4 / final curation kickoff), Mar 1 2027 (pre-orders), Apr 26 2027 (launch)

### How to import on iOS
1. Email yourself the file (or download from https://thiccctionary.com/resources/thiccctionary-roadmap.ics on your phone)
2. Tap the .ics attachment → Add All → choose your default calendar
3. Reminders fire as native iOS notifications

### How to import on Google Calendar (web or Android)
1. Open Google Calendar → Settings → Import & Export → Import
2. Upload thiccctionary-roadmap.ics → choose "Thiccctionary" calendar (create if needed)
3. Reminders fire as native Android / web notifications

### To update the schedule
Edit the .ics file, re-import. Most calendar apps will offer to merge / update existing entries.
