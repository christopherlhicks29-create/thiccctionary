## Section 8: Strategic notes

### Why we file Class 41 only, not Class 16 (printed publications) too

The book launches in April 2027. Filing Class 16 today is "intent to use" only, requires a separate Statement of Use later with another fee, and the application could mature differently from the Class 41 one. **Recommendation:** file Class 41 only now. File Class 16 as a separate application closer to book launch (~February 2027) when it's about to be use-in-commerce.

### What about the stylized wordmark?

The "The Thi**ccc**tionary" wordmark with red triple-c is a separate mark in trademark terms, a "design mark." Filing it requires submitting the visual artwork. We file standard-character first (broader), then file the design mark separately later if budget allows (~$250 additional USPTO fee).

### What happens after filing

| Stage | Timeline | What happens |
|---|---|---|
| Filing | Day 1 | Submit TEAS Plus, pay $250, receive serial number |
| Examining attorney assigned | ~3-4 months | USPTO examiner reviews. May issue an office action. |
| Office action response (if any) | Within 6 months of office action | We respond. Usually 30-90 days of work. |
| Approval for publication | ~6-9 months | If approved, mark publishes in Official Gazette |
| Publication / opposition window | 30 days | Anyone can oppose. @thiccctionaryyy could try here. |
| If no opposition | ~10-14 months from filing | Registration certificate issues. Mark is registered. |
| If opposition | TTAB proceedings | 1-2 years, may need attorney |
